## How to Deploy on GitHub Pages
1. Create a repository on GitHub (e.g., `animal-website`).
2. Upload all the files following the folder structure shown above.
3. Go to repository **Settings** → **Pages**.
4. Under "Branch", select `main` (or `master`) and `/ (root)` as the folder.
5. Click **Save**.
6. Your site will be live at `https://<your-username>.github.io/animal-website/` after a few minutes.
